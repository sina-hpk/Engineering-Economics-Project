# Engineering Economics Project

این پروژه برای تحلیل و مقایسه اقتصادی چند پروژه بر اساس مفاهیم اقتصاد مهندسی طراحی شده است.

## قابلیت‌ها
- محاسبه **NPV (ارزش فعلی خالص)**
- محاسبه **ROI (نرخ بازگشت سرمایه)**
- محاسبه **Payback Period (دوره بازگشت سرمایه)**
- مقایسه چند پروژه و انتخاب پروژه برتر
- نمایش نتایج به صورت جدول و نمودار
- پشتیبانی از تورم، مالیات و ارزش اسقاط
- رابط کاربری ساده تحت وب

## ساختار کلی
- **Backend** با Python و Flask  
  مسئول انجام محاسبات اقتصادی
- **Frontend** تحت وب  
  مسئول دریافت ورودی‌ها و نمایش نتایج
