# Engineering Economics Project

این پروژه یک ابزار ساده تحت وب برای **تحلیل و مقایسه اقتصادی پروژه‌ها** است.  
با استفاده از مفاهیم اقتصاد مهندسی می‌توان چند پروژه را با هم مقایسه و بهترین گزینه را انتخاب کرد.

## قابلیت‌ها
- محاسبه **NPV (ارزش فعلی خالص)**
- محاسبه **ROI (نرخ بازگشت سرمایه)**
- محاسبه **Payback Period (مدت بازگشت سرمایه)**
- مقایسه چند پروژه و انتخاب پروژه برتر
- رابط کاربری ساده و فارسی

## تکنولوژی‌ها
- Python
- Flask
- HTML / CSS / JavaScript
